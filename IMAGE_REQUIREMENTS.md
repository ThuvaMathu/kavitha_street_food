# Image Requirements for Kavitha Street Foods Website

## Required Images (Simple - Only 3-4 Images Needed)

You only need to create **3-4 images** for complete SEO coverage:

---

## 1. **OG Image (Open Graph / Social Media Sharing)**
   - **File Path:** `/public/og-image.jpg`
   - **Dimensions:** **1200 x 630 pixels**
   - **Format:** JPG or PNG
   - **Purpose:** Used when your website is shared on Facebook, LinkedIn, WhatsApp, etc.
   - **Content Suggestions:**
     - Your logo prominently displayed
     - Text: "Kavitha Street Foods - Authentic Indian & Sri Lankan Food"
     - Text: "Brisbane, Australia"
     - Background: Dark theme with food imagery (curry, biriyani, etc.)
     - Make it visually appealing and branded

---

## 2. **Apple Touch Icon (iOS Home Screen)**
   - **File Path:** `/public/apple-touch-icon.png`
   - **Dimensions:** **180 x 180 pixels**
   - **Format:** PNG (with transparency if needed)
   - **Purpose:** When users save your website to iPhone/iPad home screen
   - **Content Suggestions:**
     - Just your logo on a solid background
     - Simple, recognizable icon
     - Dark background with amber/gold logo works well

---

## 3. **Favicon (Browser Tab Icon)** - OPTIONAL
   - **File Path:** `/public/favicon.ico`
   - **Dimensions:** **32 x 32 pixels** (or 16x16, browser will scale)
   - **Format:** ICO or PNG
   - **Purpose:** Small icon shown in browser tabs
   - **Content:** Your logo simplified
   - **Note:** You already have `/public/logos/logo.svg` which works as favicon!

---

## 4. **Twitter Card Image** - OPTIONAL (can use same as OG)
   - **File Path:** `/public/twitter-image.jpg` (or reuse og-image.jpg)
   - **Dimensions:** **1200 x 630 pixels** (same as OG image)
   - **Format:** JPG or PNG
   - **Purpose:** When shared on Twitter/X
   - **Note:** Can use the same image as OG image to keep it simple!

---
